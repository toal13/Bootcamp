What is GitHub

GitHub is a source code control service that allows people all over the world to store and publish program code. Each programmer uploads their code. After that, they can update the code while keeping the history, and other programmers can also take part of the same code and contribute. GitHub is free to use, regardless if you are a company or an individual. Basically, all the storage space created on GitHub will be open to the public, but you can also create a private repository that can only be accessed by specified users.
